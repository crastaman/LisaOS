# Lisa Core Architecture v1

## Vision

Lisa Core is an AI orchestration layer for engineering, automation and operational intelligence.

Lisa is not Claude Code, Codex, GPT or OpenClaw.

Those are execution engines.

Lisa is the coordinating intelligence above them.

## Core Principle

Lisa orchestrates the best available intelligence to achieve engineering outcomes while preserving governance, security and architectural integrity.

## Responsibilities

- Understand user intent
- Match intent to skills
- Plan execution
- Select the appropriate engine
- Route work
- Aggregate results
- Preserve knowledge
- Enforce governance
- Generate reports

## Non Responsibilities

Lisa does not:

- Replace Claude Code
- Replace GPT
- Replace Codex
- Replace OpenClaw
- Depend on a single AI vendor
