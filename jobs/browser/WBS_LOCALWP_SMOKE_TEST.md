# Job: WBS LocalWP Smoke Test

## Capability

browser_testing
localwp_testing
qa_operator

## Target Sites

- WBS Development
- Amelia Reference
- LatePoint Reference

## Objective

Verify that the local WordPress sites load correctly and that the relevant admin areas are reachable.

## Steps

1. Open WBS Development.
2. Confirm homepage loads.
3. Open WP Admin.
4. Confirm WBS/HEB admin menu is visible.
5. Open Amelia Reference.
6. Confirm site loads.
7. Confirm Amelia admin area is reachable.
8. Open LatePoint Reference.
9. Confirm site loads.
10. Confirm LatePoint admin area is reachable.

## Output Required

- PASS / FAIL per site
- screenshots if failure occurs
- console errors
- notes
- recommended next action
